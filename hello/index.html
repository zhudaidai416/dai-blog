# 文章评论

问题：设置评论身份

解决：获取邮箱的 md5 加密值：LeanCloud ➡ 登录 ➡ 数据存储 ➡ 结构化数据 ➡ Comment ➡ 数据（mailMd5 的值）

问题：valine 获取不到评论

解决：设置 ➡ 安全中心 ➡ Web 安全域名 ➡ 添加域名即可访问

# 标签页

问题：标签页不显示下划线

解决：找到 shoka/source/css/_common/components/tags/tabs.styl ，删掉第 6 行的 overflow: hidden;
