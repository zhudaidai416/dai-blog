# 快捷键

- Ctrl + Enter：执行选中的 SQL 语句
- Ctrl + Alt + L：sql 格式化（选中需要格式化的代码）
- Ctrl + Alt + U：大小写自动转换
- 双击 Shift：全局搜索
- Ctrl + D：选择复制行
- Ctrl + N：进入某个表（按 name）
