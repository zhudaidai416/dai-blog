# commit 说明

| 类型     | 说明                         |
| -------- | ---------------------------- |
| feat     | 新功能                       |
| fix      | 修 bug                       |
| refactor | 重构，不新增功能，也不修 bug |
| docs     | 改文档，比如改 README        |
| style    | 改代码风格，不影响功能       |
| test     | 加测试、改测试               |
| chore    | 杂项，比如改 .gitignore      |
| perf     | 性能优化                     |
| ci       | CI/CD 相关改动               |
| build    | 改构建系统或依赖             |
| revert   | 回滚某个提交                 |
